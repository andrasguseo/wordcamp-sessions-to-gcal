# WordCamp Sessions to Calendar

Tampermonkey userscripts that add "Add to Google Calendar" and "Download iCal (.ics)" buttons to WordCamp schedule and session pages.

## Scripts

| Script | Event | Features |
|--------|-------|----------|
| `wceu-2026-to-gcal.user.js` | [WordCamp Europe 2026](https://europe.wordcamp.org/2026/schedule/) — Kraków, Poland | Google Calendar + iCal buttons on schedule & session pages, track prefixes, speakers in description, bulk export of favourited sessions |
| `wceu-2025-to-gcal.user.js` | [WordCamp Europe 2025](https://europe.wordcamp.org/2025/schedule/) — Basel, Switzerland | Google Calendar buttons on schedule & session pages |
| `wcus-2025-to-gcal.user.js` | [WordCamp US 2025](https://us.wordcamp.org/2025/schedule/) — Portland, Oregon | Google Calendar buttons on session pages only |

## Installation

1. Install the [Tampermonkey](https://www.tampermonkey.net/) browser extension.
2. Click on one of the script files above, then click the **Raw** button. Tampermonkey will offer to install it.

### Browser compatibility

- **Firefox**: Works out of the box.
- **Chrome / Edge / Chromium-based browsers**: You need to enable **Developer Mode** in `chrome://extensions/` and **Allow User Scripts** in Tampermonkey's extension settings. [More info.](https://www.tampermonkey.net/faq.php?q=Q209#Q209)
